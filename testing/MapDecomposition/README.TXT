README.TXT
==============================================
Function
Divide One Road Network Into N Parts with
(1) Load Balance
(2) Minimum Communication Cost
==============================================
Configuration (See 'test_configuration.txt')
--------------
(1) Road Network: Get Nodes & Sections
(2) Partition Size: N (like:2,4,5)
(3) Minimum Cut Lane Length (m): (like: 100)
(4) Cut Lane Can Curved: YES or NO
(5) Output To: 
    1 -> Console
    2 -> File
    3 -> Database
