You will simply need two use these two scripts. The input parameters are specified in the scripts.


- toy_case_gen.bash: it subsamples the demand and the fleet to run small instances of mobility service simulations. Before you run SimMobility, make sure the xml files refers to the DEMAND_PROCEDURE and FLEET_PROCEDURE specified in the script

- complete_processing.bash: it generates plots and counts. The output files generated are specified on the screen. If what is printed in the screen is too messy, just check what is written on the file. It requires octave, octave nan package and gnuplot installed
